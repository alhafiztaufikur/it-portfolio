ESP32 Monitoring, From Sensor to Dashboard

Al Hafiz Taufikur RohmanJuly 20255 min read
InputSensor reads a physical condition such as water level.
ProcessESP32 converts raw data into percentage and status.
OutputDashboard shows the latest value for quick monitoring.

The Basic Idea

An IoT monitoring system connects a physical sensor to a digital dashboard. In a water level project, the sensor reads distance, the ESP32 processes it, and the dashboard displays the condition.

Simple Data Flow

1. Sensor reading: the ultrasonic sensor measures distance from the sensor to the water surface.
2. Calculation: ESP32 converts distance into a water percentage.
3. Status logic: the percentage becomes a label such as low, normal, or high.
4. Dashboard update: data is sent to a platform such as Blynk for remote viewing.

Why Status Matters

Raw numbers are useful, but status labels are easier to understand. For example, 78% can be shown as Normal so users can react faster.

What Can Be Improved?

  • Add warning notification when water reaches a dangerous level.
  • Save historical data for trend analysis.
  • Use a waterproof enclosure for real outdoor usage.
  • Add calibration settings to improve measurement accuracy.
